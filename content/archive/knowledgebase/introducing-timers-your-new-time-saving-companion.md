<ul>
<li>
<p><strong>Summary:</strong> With the 1.2 release we will be introducing a timer feature on Ai Pin. This functionality will allow you to use the Pin to set a timer or multiple timers. For example; “set a timer for 30 minutes from now”. This article will cover how to set up and use the timer feature.</p>
<p><strong>How to use the timer feature:</strong> </p>
<ul>
<li>
<p><strong>Timer Creation:</strong></p>
<ul>
<li>
<p><span>Set a simple timer for a specified duration (e.g., "Set a timer for 10 minutes").</span></p>
</li>
</ul>
</li>
<li>
<p><strong>Timer Control:</strong></p>
<ul>
<li>Pause the last timer (e.g., "Pause the timer").</li>
<li>Resume the last timer (e.g., "Resume the timer").</li>
<li>Add time to an existing timer (e.g., "Add 10 minutes to my timer").</li>
</ul>
</li>
<li>
<p><strong>Timer Query:</strong></p>
<ul>
<li>Check how much time is left on a timer (e.g., "How much time is left on my timer").</li>
<li>View all timers (e.g., "Show me my timers").</li>
</ul>
</li>
</ul>
<p> </p>
</li>
</ul>
