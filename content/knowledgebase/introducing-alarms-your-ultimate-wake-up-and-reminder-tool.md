<p><strong>Summary:</strong> With the 1.2 release we will be introducing an alarm feature on Ai Pin. This functionality will allow you to use the Pin as an alarm clock. For example; “set a car parking alarm for 2 hours from now” or “8 am morning alarm”. </p>
<p><strong>How to set your alarm:</strong> </p>
<ul>
<li>
<strong>Voice Commands</strong>
<ul>
<li>Set Alarms: “Add new alarms” </li>
<li>Delete Alarms: “Remove existing alarms”</li>
<li>List Alarms: “Display all set alarms”</li>
</ul>
</li>
<li>
<strong>User Interface</strong>
<ul>
<li>List Alarm View: View a list of alarms with time and frequency and delete alarms using a radio button by using the phrase "Show me my alarms" and then extending your hand to invoke the laser</li>
</ul>
</li>
<li>
<strong>Offline Support</strong>
<ul>
<li>Command: "Show me my alarms" to view and delete alarms in the alarm UI.</li>
</ul>
</li>
</ul>
