<p><strong>Summary:</strong> “Something went wrong. Please try again” when using Ai Mic. This message can be caused by many things, but it typically points to an issue with the network you are connected to. If you get this message please follow the steps below.</p>
<ul>
<li>Wait 30 seconds and try again</li>
<li>Try switching to a different network (Connect to <a href="https://support.humane.com/knowledge/editor/01HMC5R7P62N5ZK695R6EYBRCJ/en-us?brand_id=13693041370765">Wi-Fi </a>or Connect to <a href="https://support.humane.com/knowledge/editor/01HMC7VXJGZQRWQT06GNRKAVGH/en-us?brand_id=13693041370765">LTE</a>)</li>
<li>Toggle Airplane Mode On/Off. Steps Found <a href="https://support.humane.com/knowledge/editor/01HMC7XNHR7GSG9ZN5V00VSNJ4/en-us?brand_id=13693041370765"><span class="wysiwyg-underline">Here</span></a>
</li>
<li>Power Ai Pin On/Off. Steps Found <a href="https://support.humane.com/knowledge/editor/01HK5Q8DXYJGJEE1FBBTQD34V6/en-us?brand_id=13693041370765"><span class="wysiwyg-underline">Here</span></a>
</li>
<li>Perform a Charge Pad Reset. Steps Found <a href="https://support.humane.com/knowledge/editor/01HXSX1PFNNBG2BM0MMNGWMD7F/en-us?brand_id=13693041370765"><span class="wysiwyg-underline">Here</span></a>
</li>
</ul>
<p><strong>Next Steps:</strong> If the troubleshooting steps above do not resolve the issue please <a href="https://support.humane.com/hc/en-us/requests/new" target="_blank" rel="noopener noreferrer">contact support</a>.</p>
